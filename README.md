## 📌 Project Overview

The **Student Management System** is a desktop-based Java application designed to help schools, colleges, and universities manage student information in an efficient and organized way.

Instead of using manual registers or spreadsheets, this system automates academic operations such as **student registration, course enrollment, grade management, and report card generation**.

👉 The main goal of this project is to **reduce administrative workload and improve accuracy in academic record management**.



## 💡 Why This Project Was Built

Managing student data manually can lead to:
- ❌ Errors in grade calculation  
- ❌ Data loss or duplication  
- ❌ Time-consuming administrative work  
- ❌ Difficulty tracking student performance  

This system solves these problems by providing a **centralized digital platform** for academic management.



## 🚀 Key Features

### 🎓 Student Registration
Easily register new students by entering personal and academic details such as name, date of birth, contact information, and enrolled courses.

### 👤 Profile Management
Administrators can view, update, and manage student profiles to keep records accurate and up to date.

### 📚 Course Management
Teachers can create courses and manage student enrollments efficiently.

### 📝 Marks Entry System
Teachers can enter student marks for different subjects in a structured way.

### 📊 Grade Calculation
The system automatically calculates final grades based on entered marks, reducing manual errors.

### 📄 Report Card Generation
Generates complete student report cards that summarize academic performance.



## 🏗️ Tech Stack

- ☕ **Java** – Core programming language  
- 🖥️ **NetBeans IDE** – Development environment  
- 🗄️ **MySQL** – Database for storing student and course data  
- 🔗 **JDBC** – Database connectivity between Java and MySQL
